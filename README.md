# Titanic_Survival_Prediction

This repository contains my solution for the famous Titanic Kaggle competition. The goal is to predict which passengers survived the tragedy.

## Performance
- *Kaggle Public Score:* 0.78229
- *Current Rank:* Top 20% of participants

## Tools & Models Used
- *Language:* Python
- *Libraries:* Pandas, Scikit-learn, XGBoost,Logistic Regression

## Key Features
- Handled missing values (Age, Fare, Embarked).
- Standardized numerical data using StandardScaler.
- Encoded categorical variables for the model.
